# Audio Sample Sources

## Folder map

- `drums/`: kick, snare, hat, clap, shaker, tom
- `piano/`: piano one-shots or short note samples
- `strings/`: string sustains, pads, stabs
- `fx/`: risers, sweeps, transition sounds
- `loops/`: optional rhythm or texture loops

## Recommended sources

- `Freesound`: safest when filtered to CC0 for raw one-shots
- `Pianobook`: good for piano and strings; check uploader responsibility note
- `Spitfire LABS`: good for composing rendered parts, not for redistributing raw library files
- `BandLab Sounds`: usable in larger compositions; do not redistribute standalone sounds
- `Pixabay`: usable for adapted content, not for standalone redistribution

## Suggested first download list

- `drums/`: kick.wav, snare.wav, hat-closed.wav, clap.wav, shaker.wav
- `piano/`: piano-c4.wav, piano-e4.wav, piano-g4.wav
- `strings/`: strings-c4.wav, strings-g4.wav, strings-pad.wav
- `fx/`: rise.wav, whoosh.wav

## Rule of thumb

- If a file will be committed and served directly by the web app, prefer `CC0`.
- If a library only allows use inside compositions, render audio from it instead of copying raw samples into this folder.
